## About

A modern, responsive portfolio website for **Musfika Rahman Ananna** - Senior Flutter Developer with 6+ years of experience in FinTech, e-learning, and logistics domains.

## Features

- Dark/Light mode toggle
- Responsive design for all devices
- Smooth scroll navigation
- Project showcase with category tabs (Banking, E-Commerce, E-Learning, Delivery)
- Publications section with research paper links
- Contact form
- App store links for published apps

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **UI Components:** shadcn/ui
- **Icons:** Lucide React
- **Theme:** next-themes

## Sections

- **Hero** - Introduction with social links
- **About** - Professional summary, skills & certifications
- **Publications** - Research paper publications
- **Services** - Core competencies
- **Projects** - Portfolio categorized by industry
- **Contact** - Contact form and information
